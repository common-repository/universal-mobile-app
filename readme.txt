Turn boring and intrusive advertising into a service for your users and maximize your revenue.

After that you can just use the shortcode to place ads in your website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/universal-mobile-ads` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Universal App (or the menu 'Universal App') screen to configure the plugin and insert your APIKEY. You can obtain or generate APIKEY from http://universal.myappfree.it
4. Activate the functions that you disire from the options (e.g. the applist) and insert the shortcode [universalapplist] anywhere in your website to show ads and generate revenue
